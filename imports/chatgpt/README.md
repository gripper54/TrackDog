# ChatGPT Import Drop Zone

Put ChatGPT exports or pasted conversations in this folder.

## Good inputs

- Full ChatGPT data export (preferred if you have lots of chats)
- Individual pasted conversations as `.md` or `.txt`
- Shared conversation page text copied into a file

## Suggested structure

- `raw/` — untouched exports or pasted dumps
- `processed/` — summaries, extracted notes, cleaned versions

## Examples

- `raw/chatgpt-export.zip`
- `raw/preferences.md`
- `raw/project-ideas.txt`

Once files are here, tell me to ingest them and I’ll extract durable memory:
- preferences
- recurring projects
- important people
- decisions
- tasks
- useful patterns

I’ll keep raw data separate from long-term memory.
